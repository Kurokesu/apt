APT Package Archive

Signed Debian packages from Kurokesu camera software.

Set up

On a 64-bit Raspberry Pi OS (Bookworm or Trixie):

curl -fsSLO https://apt.kurokesu.com/setup.sh
sudo sh setup.sh --update
sudo apt install <package>

Inspect before running. Verify the checksum first:
echo "__SETUP_SHA256__ setup.sh" | sha256sum -c
--update refreshes apt. Omit it to just enable the repo.

Manual setup (no script) 
# 1 - signing key
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://apt.kurokesu.com/kurokesu-archive-keyring.gpg \
  -o /etc/apt/keyrings/kurokesu-archive-keyring.gpg

# 2 - source (set Suites to your release: bookworm or trixie)
sudo tee /etc/apt/sources.list.d/kurokesu.sources >/dev/null <<'EOF'
Types: deb
URIs: https://apt.kurokesu.com
Suites: trixie
Components: main
Architectures: arm64
Signed-By: /etc/apt/keyrings/kurokesu-archive-keyring.gpg
EOF

# 3 - prefer Kurokesu builds
sudo tee /etc/apt/preferences.d/kurokesu.pref >/dev/null <<'EOF'
Package: *
Pin: release o=Kurokesu
Pin-Priority: 1001
EOF

sudo apt update

Signing key

kurokesu-archive-keyring.gpg. Verify it matches this fingerprint:

6385 3998 AD71 95E4 3D2D  4E83 3EBA 33E5 B464 4D7A

Remove

curl -fsSLO https://apt.kurokesu.com/setup.sh
sudo sh setup.sh --remove
